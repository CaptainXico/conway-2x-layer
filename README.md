Rule 1: Birth
A dead cell becomes alive if it has exactly three live neighbors.

Rule 2: Survival
A live cell stays alive if it has two or three live neighbors.

Rule 3: Death
Underpopulation: A live cell dies if it has fewer than two live neighbors (0 or 1)

Overpopulation: A live cell dies if it has more than three live neighbors
